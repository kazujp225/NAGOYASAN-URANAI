# MEDIA.md — 必要画像・動画一覧

このプロジェクトで使用する**外部素材（画像・動画）**の一覧と配置先を整理したファイルです。
ファイル名と配置先パスを揃えれば、ページ側のコードを書き換えずに自動で表示されます。

> 凡例：必須 = ページが成立するために必要 / 推奨 = 配置するとクオリティが上がる / 任意 = 後続フェーズで考えればOK

---

## 1. 占い一覧 サムネイル画像（必須）

配置先：`/public/fortunes/`
推奨サイズ：**1200 × 900px（4:3）/ JPEG または WebP / 各 ≤ 500KB**
用途：トップページの占いカード、`/fortunes` 一覧ページのサムネイル

| ファイル名             | 用途              |
| ---------------------- | ----------------- |

## 9. 素材の権利・利用規約

- **必ず商用利用可で、ライセンス確認済みの素材**を使用してください。
- AI生成画像を使う場合は、利用元（例：DALL-E 3、Midjourney 等）の規約と、サービスとしての配布権を確認してください。
- 占い師の顔写真には肖像権の同意書を取得してください。
- フリー素材を使う場合の推奨ソース：
  - [Unsplash](https://unsplash.com/) — 商用OK・クレジット任意
  - [Pexels](https://www.pexels.com/) — 商用OK・クレジット任意
  - [adobe stock](https://stock.adobe.com/) — 有料、商用OK
  - [iStock](https://www.istockphoto.com/) — 有料、商用OK

## 10. 画像最適化チェックリスト

- [ ] 全画像 1枚 ≤ 500KB（OGPは ≤ 1MB）
- [ ] WebP / AVIF 併用（Next.js `<Image>` が自動変換するため、現在は `<img>` 使用 → 後で差し替え）
- [ ] alt属性に意味のある説明
- [ ] Retina対応（必要に応じて 2x）
- [ ] 著作権・利用権限の確認済み

## 12. 不足時の挙動

画像を配置していない期間は、すべてのスロットに**ゴールドのSVGアイコン + "IMAGE" ラベル**のプレースホルダーが表示されます。デザインが破綻することはありません。
